# iPortfolio - Dark Mode Portfolio Template

iPortfolio is a modern, responsive, and customizable portfolio template designed for developers, designers, and freelancers. This version has been customized to support **Dark Mode** for a sleek and professional look.

## Features

- **Dark Mode Design**: The template is styled with a dark theme for better aesthetics and readability in low-light environments.
- **Responsive Layout**: Fully responsive design that works seamlessly on desktops, tablets, and mobile devices.
- **Customizable Sections**: Includes sections for Hero, About, Skills, Resume, and more.
- **Smooth Animations**: Integrated animations using AOS (Animate On Scroll) library.
- **Preloader**: A stylish preloader for better user experience.
- **Scroll-to-Top Button**: Easily navigate back to the top of the page.
- **Social Links**: Links to Telegram, GitHub, WhatsApp, and LinkedIn profiles.
- **Downloadable Resume**: A direct link to download the resume.

## Installation

1. Clone the repository or download the ZIP file:
   ```bash
   git clone https://github.com/Jesse-Jagger/iPortfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd iPortfolio
   ```
3. Open the index.html file in your browser to view the template.

---

## Usage

1. **Hero Section**: Update the hero section in index.html to include your name, title, and a brief introduction.
2. **About Section**: Add your personal details, such as birthday, phone number, city, and email, in the About section.
3. **Skills Section**: Update the skills and proficiency levels in the Skills section.
4. **Resume Section**: Replace the resume download link with your own Google Drive or hosting link.

---

## Customization

### 1. **Dark Mode Colors**
The dark mode colors are defined in the `main.css` file under the `:root` selector. You can customize the colors by modifying the following variables:

```css
:root {
  --background-color: #121212;
  --default-color: #e0e0e0;
  --heading-color: #ffffff;
  --accent-color: #bb86fc;
  --surface-color: #1e1e1e;
  --contrast-color: #ffffff;
  --nav-color: #b0b0b0;
  --nav-hover-color: #ffffff;
}
```

### 2. **Images**
Replace the placeholder images in the `assets/img` folder with your own images. Update the image paths in index.html accordingly.

### 3. **Social Links**
Update the social media links in the header section of index.html:
```html
<a href="https://t.me/your-telegram" class="telegram"><i class="bi bi-telegram"></i></a>
<a href="https://github.com/your-github" class="github"><i class="bi bi-github"></i></a>
<a href="https://wa.me/your-whatsapp" class="whatsapp"><i class="bi bi-whatsapp"></i></a>
<a href="https://www.linkedin.com/in/your-linkedin" class="linkedin"><i class="bi bi-linkedin"></i></a>
```
